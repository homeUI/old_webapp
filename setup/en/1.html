Setup - Step 1

HomeUI needs a database to work.

Please now enter the database connection credentials.

For that you open the file "config.php" in your HomeUI folder.

There you put in the Credentials

Warning:
If you want to use a dedicated user for the homeUI database, you need to create one manually.
If you are to lazy to do that, just use the user "root"

When you are done, please click the button "Check connection credentials" below.
